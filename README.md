# Radni prostor za vežbanje — Razvoj softverskih veština

Ovaj workspace se koristi za vežbanje programiranja. Mesto na kome učim, usavršavam i održavam svoje softversko-inženjerske veštine kroz praktične primere, samostalne vežbe i eksperimente.

## Fokus

- **Osnove programiranja** — sintaksa, principi, strukture podataka
- **Nove tehnologije** — Java (Spring Boot, Maven), jezici i okviri koje savladavam
- **Algoritmi i LeetCode** — rešavanje zadataka, analiza složenosti, optimizacija
- **Dizajnerski šabloni i principi** — SOLID, GoF obrasc, arhitektonske odluke
- **Sistemski dizajn** — skalabilnost, raspodeljeni sistemi, obrasci komunikacije
- **Eksperimenti i prototipovi** — testiranje ideja, brzi PoC-ovi, istraživanje novih alata
- **Beleške i reference** — dokumentovanje naučenog, lična baza znanja

## Način rada

Organizacija je slobodna i praktična — fajlovi i folderi su grupisani prema temi ili tehnologiji, bez krutih pravila. Cilj je da radni prostor bude produktivan, pregledan i da podstiče napredak.

Bez lozinki, API ključeva ili drugih osetljivih vrednosti u kodu — sve poverljive konfiguracije idu van repozitorija (environment promenljive, `.env` ili profilne konfiguracije).
